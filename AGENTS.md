# HESPERI

This project is a VitePress personal portfolio website hosted on GitHub pages.

- Web pages are in `pages/`.
- VitePress config and theme are in `.vitepress`.
- English (`pages/en/`, canonical) and simplified Chinese (`pages/zh-Hans/`) are supported.

## Commands

- `pnpm lint`: format and fix fixable lint errors (must run before `pnpm check`).
- `pnpm check`: checks types, lint, and format, doesn't change code.
- `pnpm build`: build the website.
