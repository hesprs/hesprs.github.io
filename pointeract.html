A front-end library that resolves user interactions and dispatches pan zoom events, lightweight and extensible alternative to Hammer.js.

Documentation GitHub npm

Introduction â€‹

Pointeract is a tiny JavaScript/TypeScript utility library focusing on one thing - handling user interactions with DOM elements, e.g. multitouch and touchpad.

With the unique strength of versatile typings in TypeScript and its dynamic nature, Pointeract have achieved a highly modular, extendable and efficient architecture. Its core bundle size is only 1KB minified + gzipped, functionalities come from also byte-sized modules. It's fully tree-shakable, the fewer modules you use, the smaller your bundle is.

Homepage & Documentation: pointeract.consensia.cc.
